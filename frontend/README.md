### Frontend for SHL Assessment Recommendation Engine

This is the frontend application for the SHL Assessment Recommendation Engine. It provides a user interface to input queries and view recommended assessments in tabular format.

#### Features
- Input field for user queries
- Displays recommended assessments with details such as name, type, duration, description, and test link
- Error handling and loading states

#### Technologies Used
- React
- Tailwind CSS for styling

#### Setup Instructions
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies using `npm install`
4. Create a `.env` file in the root directory refer .env_example for environment variables
5. Start the development server using `npm run dev`
